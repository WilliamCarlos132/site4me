# 开发笔记目录说明

为便于后续复刻与学习，本目录记录 ourNote 前端关键文件的编写过程与代码要点。建议阅读顺序如下：

1. `project-overview.md`：概览项目结构、全局配置步骤。
2. `app-layout.md`：主布局组件的实现细节，包括导航折叠逻辑。
3. `network-module.md`：人际关系模块（表格 & 关系图）的开发思路与 API 交互。
4. `lights-out-game.md`：熄灯游戏从 C++ 逻辑迁移到 Vue 的完整过程。

如需新增笔记，可在本目录继续创建 Markdown 文件，并更新此 README 以维持索引。
